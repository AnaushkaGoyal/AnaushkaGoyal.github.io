Under-18 Internet Access

A glyph-based map that visualizes where under-18 populations concentrate and where internet access is missing. Each centroid is rendered as a rotated radial mark, built programmatically in R by generating angles in radians and applying rotation transforms, so the pattern reads as both density and disparity.
