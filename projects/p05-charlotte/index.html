Charlotte Housing Modeling Midterm • Spatial + Regression Workflow

What this study does

This project builds a compact modeling workflow to explain housing price variation using neighborhood, accessibility, and amenity indicators. The focus is on interpretation and model quality: selecting a reasonable specification, checking assumptions, and understanding where the model performs well (and where it doesn’t).

Focus
Multivariate housing-price modeling
Methods
Feature exploration, correlation checks, OLS + diagnostics
Spatial work
Mapped predictors + neighborhood context variables
Tools
R (modeling + visualization)
